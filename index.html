2. Laburpena 2. Resumen 2. Summary

Praktika-eskulaburu honen helburua ikasleei automata programagarrien (PLC) programazioan gidatzea da, Siemens S7-1200 PLC bidez kontrolatutako didaktika-maketa pneumatiko baten bidez. Maketak automatizazio industrialaren oinarrizko kontzeptuak modu praktikoan lantzeko aukera ematen du, pneumatika, sentsorika, eskema elektrikoak eta PLC programazioa integratuz.

Maketaren Deskribapena

  • Efektu bikoitzeko bi zilindro pneumatiko.
  • Bi balbula banatzaile biestable (zilindro bakoitzarentzat bat).
  • Lau posizio-sentsore (bi zilindro bakoitzeko: retrahituta eta irekita).
  • Hiru aginte-pultsadore sistema kontrolatzeko (Martxa, Gelditu, Larrialdia) eta hautatzaile modu bat.
  • Siemens S7-1200 PLC-a, sistemaren kontrolaz arduratzen dena.
  • Seinaleztapen argidunak: Argi Berdea (Geldituta), Argi Gorria (Zikloa martxan) eta Argi Laranja (Eskuzko modua).
  • Argi osagarria: Gaueko fokua maketa hobeto ikusteko.

Este manual de prácticas tiene como objetivo guiar al alumnado en el aprendizaje de la programación de autómatas programables (PLC) mediante el uso de una maqueta neumática didáctica, controlada por un PLC Siemens S7-1200. La maqueta permite trabajar de forma práctica conceptos fundamentales de automatización industrial, integrando neumática, sensórica, esquemas eléctricos y programación PLC.

Descripción de la Maqueta

  • Dos cilindros neumáticos de doble efecto.
  • Dos válvulas distribuidoras biestables (una para cada cilindro).
  • Cuatro sensores de posición (dos por cilindro: totalmente retraído y totalmente extendido).
  • Tres pulsadores de mando para el control del sistema (Marcha, Paro, Emergencia) y selector de modo.
  • PLC Siemens S7-1200, encargado del control global del sistema.
  • Pilotos de señalización: Luz Verde (Parado), Luz Roja (Funcionamiento) y Luz Naranja (Modo Manual).
  • Iluminación auxiliar: Foco nocturno incorporado para baja visibilidad.

The main objective of this lab manual is to guide students in learning programmable logic controller (PLC) programming through a training pneumatic setup controlled by a Siemens S7-1200 PLC. The training model enables hands-on practice of basic industrial automation concepts by integrating pneumatics, sensors, electrical diagrams, and PLC programming.

Model Description

  • Two double-acting pneumatic cylinders.
  • Two bistable directional control valves (one per cylinder).
  • Four position sensors (two per cylinder: fully retracted and fully extended).
  • Three control pushbuttons (Start, Stop, Emergency) and a mode selector switch.
  • Siemens S7-1200 PLC, responsible for the global system control.
  • Status indicator lights: Green Light (Stopped), Red Light (Running), and Orange Light (Manual Mode).
  • Auxiliary lighting: Integrated Night spotlight for low-visibility environments.

Lan-modu hautatzaile bat integratu (`I0.5`).

Integrar un selector de modo de trabajo (`I0.5`).
